・ファイルの説明
　　Shooting.exe  windows用の実行ファイル
　　./Shooting    ゲームエンジンUnity用のプロジェクトファイル


・作品内容
　　２Ｄのシューティングゲームです。
　　キーボードのカーソルキーで移動、左のCtrlキーで撃ちます。

・制作期間
　　約１０日です。

・開発環境
　　ゲームエンジンUnityを使って開発しました。
　　Unity 公式HP: www.unity3d.com

・工夫した点
　　敵の動きや弾の速さを調節して、難易度がちょうどよくなるよう工夫しました。
　　敵のよける動きが独特になるようパラメータやアルゴリズムなどを工夫しました。
　　開発の際、サイトの情報を参考にしましたが、いろいろなサイトを見比べて、自分
　　なりに分かりやすいコードになるように気を付けました。
